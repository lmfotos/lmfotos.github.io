## Installation

1. Clone the repository:
    ```sh
    git clone https://github.com/yourusername/lm-website.git
    ```
2. Navigate to the project directory:
    ```sh
    cd lm-website
    ```
3. Install the dependencies:
    ```sh
    npm install
    ```

## Usage

To start the development server, run:
```sh
npm start
```
